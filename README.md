# Sample Android App - Countries Recyclerview

## Tech Stack

- Architecture Pattern: MVVM
- UI Framework: XML
- Remote Data Access: Retrofit
- Testing: Local 

**Libraries:**
* Kotlin Coroutines
* Gson
* Retrofit
* Testing- Junit-4 & Mockito
